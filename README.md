# Конфигурация 1С

Это файловая выгрузка конфигурации 1С.

## Структура

- Catalogs — справочники
- Documents — документы
- Languages — языки
- Configuration.xml — корневое описание конфигурации

## Важно

Не хранить в репозитории:

- выгрузки базы *.dt
- бинарные конфигурации *.cf
- пароли
- токены
- персональные данные
